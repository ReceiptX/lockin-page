What is LockIn?

LockIn is a production-ready, client-side JavaScript SaaS solution that allows you to protect any webpage with a password-protected modal overlay. Perfect for:

  • Private content pages
  • Beta testing environments
  • Exclusive member areas
  • Preview pages for clients
  • Development/staging environments

Key Features

  • Single script tag integration
  • No server-side code required
  • Fully configurable via data attributes
  • Session-based authentication
  • Optional branding control
  • Mobile responsive design
  • Smooth animations
  • Clean, production-ready code

Installation

Add LockIn to any webpage with a single script tag:

<script src="https://your-cdn.com/lockin.js" data-lockin-key="YOUR_LICENSE_KEY" data-lockin-password="YOUR_PASSWORD"></script>

Configuration Options

Attribute Description Required
data-lockin-key Your license key Yes
data-lockin-password Access password Yes
data-lockin-title Modal title No
data-lockin-message Modal message No
data-lockin-branding Show branding (use "hide" to remove) No
